# Java Swing Calculator

Calculadora feita em Java utilizando Swing.

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Números negativos
- Delete
- Clear

## Tecnologias

- Java
- Swing
- JFrame
- ActionListener

## Objetivo

Projeto criado para estudos de Java Desktop e interfaces gráficas.
